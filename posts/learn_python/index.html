Generators#

let’s consider the function below:

import time

def count():
    time.sleep(1)

and you want to change the behavior of it so it’s runs and write the time of the execution of it.

that could be done in several ways

way 1:#

import time

def count():
    time.sleep(3)

t0 = time.time()
count()
t1 = time.time()
print(t1 - t0)

3.0073814392089844

let’s make that better by putting all into a function
you might know that a function name in python is just a referance to it so we can pass that as an argument

import time

def count():
    time.sleep(3)

def count_ex_time(fct):
    t0 = time.time()
    fct()
    t1 = time.time()
    return t1 - t0

print(count_ex_time(count))

but now instead of calling the function count_ex_time to count time of execution and running it why not change the behaviour of the function count itself so it will run as before and also calculate the time of execution, we can achieve that by assigning the new function that we build to the count function, think of it like we change the name of the second function to the first function name, and loosing referance to the first one as we are no longer need it.

import time

def count():
    time.sleep(3)

def count_ex_time(fct):
    t0 = time.time()
    fct()
    t1 = time.time()
    return t1 - t0

count = count_ex_time() # change the name of the new function

print(count())
